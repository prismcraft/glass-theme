# Quickstart

## 1) Install deps

```bash
pnpm install
```

## 2) Build packages

```bash
pnpm build
```

## 3) Run examples

```bash
pnpm --filter glass-example-vanilla dev
pnpm --filter glass-example-react dev
```

## 4) Publish (optional)

Log into npm as `mgrigajtis` before running the publish steps.
